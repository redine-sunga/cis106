# What is Markdown?
- Markdown is a *markup language* that let's you write in plain text documents with a few lightweight formatting options.

# What is Git?
- Git is a *version control system*.
- It controls and tracks different versions of a given project.

# What is GitHub?
- [GitHub](https://docs.github.com/en/get-started/start-your-journey/about-github-and-git) is a **cloud-based** platform where you can store, share, and work together with others to write code.
- It stores your code in *repositories*
- Since Github is cloud-based, it is hosted by Git.

# What is Slack?
- [Slack](https://slack.com/help/articles/115004071768-What-is-Slack) is the operating system for work. By bringing together people, processes, data, agents, and AI into one conversational interface, Slack transforms how organizations accomplish their goals.
